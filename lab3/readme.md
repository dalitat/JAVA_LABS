Лабораторная работа №3

В данной лабораторной работе изучаются принципы работы хэш-таблиц,
способы разрешения коллизий методом цепочек, а также применение
встроенного класса HashMap.

Реализована собственная обобщенная хэш-таблица HashTable<K, V> с
методами put(key, value), get(key), remove(key), size() и isEmpty().
Коллизии разрешаются методом цепочек с использованием LinkedList.
Для вычисления индекса используется хэш-функция на основе hashCode() ключа.

Также выполнено задание со встроенным HashMap: создан класс Car с полями
brand, model, year, и класс CarFleet, использующий HashMap<String, Car>
для учета автомобилей в автопарке. Реализованы операции добавления,
поиска, удаления автомобиля по номерному знаку и вывод всех записей.

В классе Main продемонстрирована работа обеих частей задания.
В результате освоены принципы организации хэш-таблиц и работы с
коллекциями типа «ключ-значение».
