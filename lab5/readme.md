Лабораторная работа №5

В данной лабораторной работе изучаются регулярные выражения в Java
с использованием классов Pattern и Matcher.

Реализована программа NumberFinder для поиска всех целых и дробных чисел
в тексте.

Реализована программа PasswordValidator для проверки пароля: только
латинские буквы и цифры, длина от 8 до 16 символов, хотя бы одна заглавная
буква и одна цифра.

Реализована программа LowercaseUppercaseFinder для поиска случаев, когда
после строчной буквы сразу идет заглавная, и выделения таких фрагментов
символами ! с двух сторон.

Реализована программа IPAddressValidator для проверки корректности
IP-адреса: 4 числа от 0 до 255, разделенных точками.

Реализована программа WordFinder для поиска всех слов, начинающихся с
заданной буквы (без учета регистра).

В результате освоены регулярные выражения для поиска, проверки и
обработки текстовой информации.
