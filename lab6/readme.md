Лабораторная работа №6

В данной лабораторной работе изучаются коллекции в Java, дженерики и
итераторы.

Реализована программа TopWords, которая считывает текстовый файл и
выводит топ-10 самых часто встречающихся слов с использованием
HashMap<String, Integer> для подсчета частоты и сортировки через
Collections.sort().

Реализован обобщенный класс Stack<T> на основе массива с методами
push(), pop(), peek(), size(), isEmpty(), isFull(). Стек выбрасывает
исключения при переполнении и попытке извлечения из пустого стека.

Для задания 3 (вариант 9) реализована программа учета продаж с
использованием ConcurrentHashMap для потокобезопасного хранения пар
«товар-количество продаж» и AtomicInteger для счетчика продаж.
Созданы классы ProductSales (с полями name, price, AtomicInteger count)
и SalesTracker с методами addSale(), printSales(), getTotalSum(),
getMostPopularProduct(). В классе SalesDemo продемонстрирована работа
всех методов.

Все три задания выполнены в полном объеме.
