Лабораторная работа №2

В данной лабораторной работе изучаются основные концепции
объектно-ориентированного программирования в Java: абстракция,
инкапсуляция, наследование и полиморфизм.

Реализована иерархия классов по варианту 4 (транспорт). Создан абстрактный
класс Transport с полями model, speed, fuel (модификатор protected),
конструкторами, геттерами и сеттерами, статическим полем count для
подсчета объектов, абстрактным методом showInfo() и перегруженным
методом move().

Созданы дочерние классы: Car (автомобиль) с полями bodyType и doors,
Truck (грузовик) с полями loadCapacity и hasTrailer, Motorcycle
(мотоцикл) с полями handlebarType и hasSidecar. В каждом классе
переопределен метод showInfo().

В классе Main созданы объекты всех типов, продемонстрирована работа
геттеров и сеттеров, перегрузка метода move(), полиморфизм через массив
Transport[], а также статический счетчик count.

Все принципы ООП реализованы в полном объеме.
