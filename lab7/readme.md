Лабораторная работа №7

В данной лабораторной работе изучаются основы многопоточности в Java.

Реализована программа Task1_SumArray, вычисляющая сумму элементов массива
с использованием двух потоков: каждый поток обрабатывает свою половину
массива, результаты суммируются в главном потоке с помощью join().

Реализована программа Task2_MatrixMax для поиска наибольшего элемента
в матрице: каждый поток обрабатывает одну строку, находит максимум в
строке, затем главный поток сравнивает результаты.

Для задания 3 (вариант 9) реализована программа моделирования работы
грузчиков на складе с использованием CyclicBarrier. Созданы класс
Product (name, weight), класс Warehouse с очередью товаров и ограничением
веса 150 кг, класс Loader (наследник Thread), который переносит товары
до достижения лимита, затем вызывает barrier.await() для синхронизации
трех грузчиков.

В методе main создаются 6 товаров разного веса, три грузчика и
CyclicBarrier с действием после достижения барьера.

Продемонстрирована работа всех трех заданий.
